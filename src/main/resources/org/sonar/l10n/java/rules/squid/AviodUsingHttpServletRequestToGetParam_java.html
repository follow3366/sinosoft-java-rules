不允许使用HttpServletRequest 对象的方法获取参数值。

在任何方法中不允许直接调用 javax.servlet.http.HttpServletRequest 对象的方法获取参数值，必须先创建 com.sinosoft.sysframework.common.util.ParamUtils 对象。

不合规范代码示例

package com.sinosoft.claim.ui.control.facade;

public String UITestFacade extends Action{

    public String execute(HttpServletRequest request) {

    String codeType = (String)request.getAttribute("codeType"); // Noncompliant

    return codeType;
    }
}

合规代码示例

package com.sinosoft.claim.ui.control.facade;

public String UITestFacade extends Action{

    public String execute(HttpServletRequest request) {

    ParamUtils paramUtils = new ParamUtils(request);
    String codeType = paramUtils.getParameter("codeType"); // Compliant

    return codeType;
    }
}
